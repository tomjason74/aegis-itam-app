# Aegis Health Partners — ITAM Intelligence Platform

Enterprise IT Asset Management system built for Aegis Health Partners.
Automated provisioning, HIPAA compliance, and predictive lifecycle intelligence.

## Tech Stack
- Node.js + Express (Backend)
- WebSocket (Real-time event streaming)
- Vanilla HTML/CSS/JS (Frontend SPA)
- In-memory simulated PostgreSQL, MongoDB, InfluxDB

## Start Locally
```bash
npm install
npm start
```
Open http://localhost:3000
